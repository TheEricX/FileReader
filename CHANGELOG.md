# Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog, and this project follows a date-based versioning scheme.

## [2026-01-26]
### Added
- Excel viewer: configurable frozen rows/columns and row numbers.
- Excel viewer: range selection with a reference prompt for the chat input.
- Chat: reference banner in the input area.
- Chat: draggable divider to resize message area vs input area.

### Changed
- Chat input: multi-line input with Enter to send and Shift+Enter for newline.
- UI: selection highlight styling and input layout adjustments.
